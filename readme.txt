Onlineshop - made easy. Noch nie war es so einfach Ihre Wordpress Webseite durch einen Onlineshop zu erweitern. Einfach das kostenlose Plugin installieren und &uuml;ber Ihre Wordpress Seite all Ihre Produkte und Artikel online zum Verkauf anbieten!

Durch die nahtlose Verbindung zwischen Ihrer WordPress Website und Ihrem Online-Shop powered by brodos.net bieten Sie Ihren Kunden den optimalen Mix aus Image, Content und Shoppingerlebnis. 

== Highlights ==
* Online-Shop-Suchfunktion | Artikel auf Ihrer Website direkt suchen und finden
* Warenkorbfunktion | Warenkorb auf Ihrer Website fix integrieren
* Kategorieintegration | Shopkategorien auf Ihrer Website prominent platzieren
* Integration individueller Produktgruppen | Highlight Produkte auf Ihrer Website gezielt promoten
* Direkte Artikelverlinkung | Top-Artikel auf Ihrer Website aktuell anpreisen
* Gratis Plugin | alle Funktionen for free

Mit dem brodos.net Plugin machen Sie aus Ihrer Wordpress-Webseite eine voll funktionsf&auml;hige E-Commerce Plattform.

Voraussetzung f&uuml;r die Nutzung sowie Funktionsf&auml;higkeit des WordPress Plugins&nbsp; ist ein mit Ihrem Sortiment best&uuml;ckter Onlineshop powered by brodos.net. 

Mehr Informationen finden Sie unter [brodos.net/onlineshop](https://brodos.net/omnichannel/onlineshop)
